# JetBrains Tab Manager

Smart Tab Manager plugin for JetBrains IDEs (IntelliJ IDEA, GoLand, PyCharm, RustRover, WebStorm).

## Features (Coming Soon)
- **Tab Grouping** — Select and group tabs together with labels
- **Tab Coloring** — Assign colors to tabs for visual distinction
- **Workspaces** — Save/restore collections of open tabs and tool windows
